### 🧩 Hybrid AI Log Classification System
**FastAPI • LangChain • Google Gemini • Sentence Transformers • scikit-learn • Pandas**

A multi-stage log intelligence platform that combines rule-based parsing, semantic embeddings, and LLM reasoning to automate enterprise log classification.

**Highlights**
- Hybrid routing across deterministic and AI-based classification paths
- FastAPI service layer for inference and orchestration
- Semantic similarity using transformer embeddings
- Built to handle log complexity with structured fallback logic

[Repository](https://github.com/arnavmaheshwari/Log-Classification-System)

### 🩺 Retrieval-Augmented Medical Chatbot
**Flask • LangChain • Google Gemini • Pinecone • HuggingFace Embeddings • Docker • GitHub Actions**

A retrieval-augmented chatbot that answers medical questions using document ingestion, semantic retrieval, and vector search grounded in uploaded content.

**Highlights**
- RAG pipeline with document chunking and retrieval
- Vector search over embedded medical content
- LLM-backed contextual answers using Gemini
- Containerized deployment workflow with CI/CD support

[Repository](https://github.com/arnavmaheshwari/Medical-Chatbot)
